# Bee-Bot PWA

Simulador Bee-Bot instalable (PWA) con funcionamiento offline.

## Project info

Este proyecto es una aplicación web progresiva (PWA) construida con React y Vite, diseñada para simular un Bee-Bot.

## Development

Para ejecutar el proyecto localmente:

1.  Clona el repositorio.
2.  Instala las dependencias:
    ```sh
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
-   Vite PWA Plugin
